# TODO - Evo-AI 项目

## 📋 项目概述

Evo-AI 是一个自主 AI 监督系统，采用 Master-Slave 架构，Master AI 协调多个 Slave AI agents 完成软件开发任务。

- Git worktree 用于隔离执行环境
- 配置文件支持：`.evo-ai/config.json` 和 `.evo-ai/credentials.json`
- 状态持久化：`.evo-ai/.data/`
